# RedRainDemo


RedRainDemo是基于https://www.jianshu.com/p/629e9971c48b实现的展示型红包雨APP。该分支针对原生新版食物派开发，主要针对倒计时开始红包雨任务（仍未完全迁移，所以存在部分Redux代码，如有兴趣可与我联系），不做商业用途。目前已适配。如有问题和改进意见，欢迎提issue一起讨论学习。
